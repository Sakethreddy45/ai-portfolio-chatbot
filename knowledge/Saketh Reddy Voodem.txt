Project: AI Portfolio Chatbot

Built a chatbot that works as an interactive portfolio where visitors can ask anything about Saketh's background and get real answers. Under the hood it runs a RAG pipeline. Resume and project docs get chunked and embedded using OpenAI's text-embedding-3-small model then stored in ChromaDB. When someone asks a question the system embeds their query finds the closest matching chunks via cosine similarity and feeds that context into GPT-4o-mini to generate a response. The bot only talks about things Saketh has actually done. The backend is FastAPI with SQLite tracking knowledge entries, conversation logs, visitor leads, and unanswered questions. There is a password-protected admin panel for uploading new documents, adding specific Q&A pairs, viewing what people are asking, and seeing who left their email. OpenAI function calling handles the lead capture and logs questions the bot could not answer. Pushover notifications send real-time alerts when someone drops their email or asks something the bot does not know. Deployed on Hugging Face Spaces. Tech stack includes Python, FastAPI, OpenAI API, ChromaDB, SQLite, RAG, Prompt Engineering, OpenAI Function Calling, HTML, CSS, JavaScript, Docker, Hugging Face.
